matplotlib
----------

`apt-get install python-matplotlib`


notebook
-------
`apt-get install ipython-notebook`

`ipython notebook`

用pylab模式运行notebook(自动加载Scipy Numpy Matplotlib模块)

`ipython notebook --pylab`

使用inline指令可以在notebook的单元格中显示嵌入式图表

`ipython notebook --pylab inline`
